Examen Git y Github

Alumn@: Cesar Justo

  1. Que es un sistema de control de versiones?

    Un sistema de control de versiones te permite mantener un historico en el tiempo de todas las versiones asociado a tu codigo fuente, pudiendo retroceder en el tiempo a cualquier version que se encuentre dentro del mismo.

  2. Cual es la diferencia entre git y github?

    Git es un sistema de control de versiones y Github es un servicio en el cual puede almancenar tu codigo y tener un historico en tiempo de las versiones del mismo mediante git.

  3. Cómo creo una rama en git?

    Puedes crear una rama con el comando git checkout -b nombre_rama

  4. Cómo clono un repositorio?

    Se puede clonar un repositorio remoto mediante el comando git clone
